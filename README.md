# Python OOPS Examples

This repository contains clean, beginner-friendly Python programs
demonstrating core Object-Oriented Programming concepts.

## Files
1. inventory_oops.py – Encapsulation, Inheritance, Polymorphism
2. composition_supplier_product.py – Composition & Exception Handling
3. employee_manager_inheritance.py – Protected Attributes & Overriding
4. abstraction_methods_properties.py – Abstraction, Static/Class Methods, Properties
